JARVIS CHAT UI POLISH PACK 5

Что делает:
- убирает список файлов библиотеки из левого сайдбара
- убирает кнопку загрузки файлов слева
- все файлы библиотеки показывает в основном окне
- включает автоскролл на новые сообщения агента
- уменьшает шрифт сообщений и поля ввода примерно на 15%
- действия чата сверху переводит в иконки:
  - память
  - закреп
  - переименовать
  - удалить
- убирает функции из левого списка чатов, там остаётся только список
- делает автопереименование новых чатов по первому пользовательскому сообщению
- длинные названия в левом списке показываются через ellipsis
